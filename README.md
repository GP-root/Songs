# songs

## 运行

### 1.安装依赖

```
npm install
```

### 2.编译

```
npm run serve
```

### 3.打包

```
npm run build
```

### 4.查看项目文档

本地浏览（需安装docsify）

```
docsify serve docs
```

线上浏览

```
https://gp-root.github.io/Songs/#/
```
