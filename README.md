# songs


## 基本介绍

该项目是一个音乐播放网站，后端数据使用的是部署在我的服务器上的[网易云音乐API](https://github.com/Binaryify/NeteaseCloudMusicApi)。

使用到的技术：

- vue3
- vue-router
- pinia
- axios
- element-plus
- TypeScript
- ...

## 项目运行

1.安装依赖

```
npm install
```

2.编译

```
npm run serve
```

3.打包

```
npm run build
```

4.查看项目文档

```
docsify serve docs
```
